## Design why-trail

| Choice | Over | Because |
|---|---|---|
| Velostat primary | piezo, FSR | piezo saturated from jogging; FSR works but 18mm dot. Velostat = FSR-but-any-size |
| 1Ω + 22Ω limiter | 10kΩ stock | bench-tuned hard-hit gate; 22Ω caps short-current ~140mA vs ~3A |
| PUL heat-weld | sewn, laminator | fused = no needle holes, immersion-grade |
| PETG strike plate | bare stack | keeps face flat on torso, spreads point hits |
| Snap-fastener leads | solder-to-fabric | mechanical clamp; solder cracks |
| **M8 socket through PETG tab** | gland+pigtail | same flange/nut mounting; cable swappable both ends |
| E6000 PUL→tab before socket | clamp-only | PUL can't slip from under O-ring |
| Grommets in welded margin | loops-in-weld, bag | solid margin = no leak path; mech-certain |
| Pelican 1010 clear lid | junction box | latch = reset/charge, 2 holes only |
| Ring dark-until-hit | always-on | ~45h battery; opponents see HP only when they earn it |
| Buckle harness | hooks, JPC | no sewing, click-swap pads |
| Perfboard #1 → PCB fleet | all-perfboard | 30 hand-wired = 25h + 30 bug chances |
